EliteCart - Premium E-Commerce Store
Live Demo: https://mariajiju26.github.io/EliteCart/

Project Overview
EliteCart is a modern, responsive Static Site developed as a comprehensive e-commerce platform. It features a clean, professional user interface designed to showcase products across Electronics, Fashion, and Home Decor categories.

Core Features
Interactive Product Navigation: Each product card includes a "View" or "Details" button that redirects to product-view.html using unique ID parameters (e.g., ?id=elec5) for specific item data.
Dynamic Search Functionality: A prominent search bar in the sticky navbar allows users to query the catalog via search.html.
Responsive Department Gallery: The "Shop by Department" section uses high-resolution imagery and hover-ready cards to categorize the shopping experience.

Trending Highlights: A specialized "Trending Now" grid displays top-rated items with clear currency formatting (₹) and high-quality photography.
Sticky Navigation: Implements sticky-top classes to keep the navbar accessible during long scrolls.
Social Integration: Footer includes stylized FontAwesome icons for brand social presence on Facebook, Instagram, and Twitter.

Front-End Technology Stack
HTML5: Used for semantic page structure and content organization.
CSS3: Custom styling for layout, typography, and unique branding elements.
Bootstrap 4.5: Core framework utilized for the mobile-first grid system and responsive components.
JavaScript: Handles client-side interactivity and basic navigation logic.
FontAwesome: Integrated for professional iconography throughout the site.

File Structure
index.html: The primary landing page and entry point (renamed from home.html for hosting compatibility).
products.html: Full catalog display for all items.
product-view.html: The dynamic template for individual product details.
style.css: Contains custom overrides for Bootstrap and unique UI design.

Deployment
Hosting: Deployed via GitHub Pages (and Render) for 24/7 availability.

Workflow: Integrated with GitHub Desktop for continuous version control and seamless updates to the main branch.
